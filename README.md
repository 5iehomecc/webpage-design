# 🌍 网页设计 & HTML小游戏合集

本仓库包含多个精美的网页应用和HTML小游戏，均采用单文件实现，无需额外依赖，可直接在浏览器中运行。

---

## 🎮 HTML小游戏合集

### 经典扫雷 (minesweeper.html)
完美复刻Windows经典扫雷游戏
- **操作**: 左键点击揭开，右键标记地雷，双键验证数字范围
- **特性**: 三种难度（初级/中级/高级）、计时器、地雷计数
- **配色**: 经典Windows 98格子配色 + Happy Hues Palette 10外框

[预览扫雷](https://5iehome.cc/minesweeper.html)

### 纸牌接龙 (solitaire.html)
经典Windows XP风格Klondike纸牌接龙
- **操作**: 拖拽移动、双击自动移入基础堆/牌列
- **特性**: 发牌堆Draw1/Draw3模式、计时器、计分、胜利自动播放
- **配色**: 绿色牌桌 + Windows XP窗口风格

[预览纸牌接龙](https://5iehome.cc/solitaire.html)

### 小游戏合集主页 (games.html)
HTML小游戏合集展示页
- **风格**: Happy Hues Palette 10 配色方案
- **特色**: FC/NES经典游戏人物8-bit像素动画装饰

[预览合集](https://5iehome.cc/games.html)

---

## ⏰ 时区转换 (timezone-converter.html)

优雅的苹果风格时区转换工具
- **特性**: 多城市支持、自动DST识别、昼夜视觉区分
- **支持城市**: 北京、马德里、赫尔辛基、苏黎世、密尔沃基

[预览时区转换](https://5iehome.cc/timezone-converter.html)

## 📂 项目结构

```
/workspace/

├── minesweeper.html            # 经典扫雷游戏

```

## 🎨 UI 设计

- **主题色**：苹果蓝 (#007aff)
- **昼夜切换**：白天使用太阳图标，夜间使用月亮图标
- **卡片高亮**：基准城市使用金黄色边框 (#FFD54F) 高亮
- **国旗色彩**：每个城市使用对应国旗的官方颜色
