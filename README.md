# 🌍 网页设计 & HTML小游戏合集

本仓库包含多个精美的网页应用和HTML小游戏，均采用单文件实现，无需额外依赖，可直接在浏览器中运行。

---

## 🎮 经典扫雷 (minesweeper.html)

完美复刻Windows经典扫雷游戏
- **操作**: 左键点击揭开，右键标记地雷，双键验证数字范围
- **特性**: 三种难度（初级/中级/高级）、计时器、地雷计数
- **配色**: 经典Windows 98格子配色 + Happy Hues Palette 10外框

[预览扫雷](https://5iehome.cc/minesweeper.html)

---

## ⏰ 全球时区转换 (timezone-converter.html)

优雅的苹果风格时区转换工具
- **特性**: 多城市支持、自动DST识别、昼夜视觉区分
- **支持城市**: 北京、马德里、赫尔辛基、苏黎世、密尔沃基

[预览时区转换](https://5iehome.cc/timezone-converter.html)
